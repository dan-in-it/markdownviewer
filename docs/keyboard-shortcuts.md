# Markdown Editor Keyboard Shortcuts

A comprehensive shortcut reference organized by category.

## Text Formatting

| Shortcut | Action |
| --- | --- |
| `Ctrl+B` | Bold |
| `Ctrl+I` | Italic |
| `Ctrl+U` | Underline |
| `Ctrl+Shift+S` | Strikethrough |
| `Ctrl+\`` | Inline code |
| `Ctrl+Shift+H` | Highlight/mark |

## Headings

| Shortcut | Action |
| --- | --- |
| `Ctrl+1` | Heading 1 |
| `Ctrl+2` | Heading 2 |
| `Ctrl+3` | Heading 3 |
| `Ctrl+4` | Heading 4 |
| `Ctrl+5` | Heading 5 |
| `Ctrl+6` | Heading 6 |
| `Ctrl+0` | Clear/normal paragraph |

## Lists

| Shortcut | Action |
| --- | --- |
| `Ctrl+Shift+8` | Unordered (bullet) list |
| `Ctrl+Shift+7` | Ordered (numbered) list |
| `Ctrl+Shift+C` | Task/checkbox list |
| `Tab` | Indent list item |
| `Shift+Tab` | Outdent list item |

## Blocks & Structure

| Shortcut | Action |
| --- | --- |
| `Ctrl+Shift+K` | Code block (fenced) |
| `Ctrl+Shift+Q` | Blockquote |
| `Ctrl+Shift+R` | Horizontal rule |
| `Ctrl+Shift+T` | Insert table |
| `Ctrl+Enter` | Insert line break / new block below |

## Links & Media

| Shortcut | Action |
| --- | --- |
| `Ctrl+K` | Insert/edit link |
| `Ctrl+Shift+I` | Insert image |
| `Ctrl+Shift+V` | Paste as plain text |
| `Ctrl+V` | Smart paste (auto-linkify URLs over selected text) |

## Selection & Navigation

| Shortcut | Action |
| --- | --- |
| `Ctrl+A` | Select all |
| `Ctrl+D` | Select word / select next occurrence |
| `Ctrl+L` | Select line |
| `Ctrl+Shift+Up` | Move line up |
| `Ctrl+Shift+Down` | Move line down |
| `Alt+Up` | Scroll up |
| `Alt+Down` | Scroll down |
| `Ctrl+G` | Go to line |
| `Ctrl+Home` | Go to beginning of document |
| `Ctrl+End` | Go to end of document |
| `Ctrl+Shift+\\` | Jump to matching bracket |

## Find & Replace

| Shortcut | Action |
| --- | --- |
| `Ctrl+F` | Find |
| `Ctrl+H` | Find and replace |
| `F3` / `Enter` | Find next |
| `Shift+F3` | Find previous |
| `Alt+Enter` | Replace all |
| `Ctrl+Shift+F` | Search across files (if multi-file) |

## Editing

| Shortcut | Action |
| --- | --- |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` / `Ctrl+Shift+Z` | Redo |
| `Ctrl+X` | Cut line (no selection) / Cut selection |
| `Ctrl+C` | Copy line (no selection) / Copy selection |
| `Ctrl+Shift+D` | Duplicate line/selection |
| `Ctrl+Backspace` | Delete word left |
| `Ctrl+Delete` | Delete word right |
| `Ctrl+Shift+K` | Delete line |
| `Ctrl+/` | Toggle comment (HTML comment) |
| `Ctrl+J` | Join lines |

## View & Interface

| Shortcut | Action |
| --- | --- |
| `Ctrl+Shift+P` | Command palette |
| `Ctrl+\\` | Toggle sidebar/file explorer |
| `Ctrl+Shift+M` | Toggle preview pane |
| `Ctrl+Shift+E` | Switch between editor/preview/split |
| `F11` | Toggle fullscreen / distraction-free mode |
| `Ctrl+=` / `Ctrl++` | Zoom in |
| `Ctrl+-` | Zoom out |
| `Ctrl+Shift+0` | Reset zoom |
| `Ctrl+Shift+W` | Toggle word wrap |
| `Ctrl+Shift+L` | Toggle line numbers |

## Table Editing (when cursor is in a table)

| Shortcut | Action |
| --- | --- |
| `Tab` | Move to next cell |
| `Shift+Tab` | Move to previous cell |
| `Enter` | Move to next row |
| `Ctrl+Shift+Enter` | Add row below |
| `Ctrl+Shift+Backspace` | Delete row |
| `Alt+Shift+Right` | Add column right |
| `Alt+Shift+Left` | Add column left |
